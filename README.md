# HR Analytics Dashboard

## Project Overview

The HR Analytics Dashboard is an interactive Power BI dashboard developed to analyze employee data and workforce trends.

The dashboard helps understand employee distribution, salary patterns, satisfaction levels, department analysis, and attrition trends.

## Dataset Information

**Dataset Name:** HR Analytics Employee Dataset

The dataset contains:

* Employee ID
* Name
* Age
* Department
* Job Role
* Salary
* Experience Years
* Attrition
* Satisfaction Score

## Tools Used

* Power BI Desktop
* Excel Dataset
* Data Cleaning
* Data Visualization
* Data Analysis

## Dashboard Features

### 1. Total Employees KPI

Shows the total number of employees in the organization.

### 2. Department-wise Employee Analysis

Displays employee distribution across departments.

### 3. Attrition Analysis

Analyzes employees who stayed and employees who left.

### 4. Salary Analysis

Shows salary distribution among departments.

### 5. Employee Satisfaction Analysis

Displays employee satisfaction levels.

## Key Insights

* Employee distribution can be analyzed department-wise.
* Attrition patterns help understand employee retention.
* Salary analysis provides workforce compensation insights.
* Satisfaction analysis helps improve employee engagement.

## Conclusion

The HR Analytics Dashboard provides valuable insights into employee demographics, satisfaction, salary patterns, and attrition trends. It supports HR teams in making data-driven decisions.
